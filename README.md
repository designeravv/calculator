# Floss-calculator
Калькулятор мулине помогает разложить бленды на отдельные цвета, автоматически добавляя длину к уже существующим.  Важно, можно загрузить информационный файл из WinStitch и получить четкую таблицу из 2 столбцов Номер цвета и сколько метров потребуется для дизайна. Экономьте время на подготовке схем или подготовке заказа мулине в магазин.
